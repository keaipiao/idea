## 3. 已知风险 / 不测项

- **Testcontainers Windows + Docker Desktop WSL2 backend 不识别 `dockerDesktopLinuxEngine` named pipe**(踩坑 D2)。当前用本机 shared dev PG + `@Transactional` rollback 代替。
  - 风险:测试 + dev 同 PG 实例,sequence 推进(`t_user_id_seq` 增长)但不影响业务
  - 后续 PR-2/3 启动时若环境改善(或加 Linux CI),可切回 Testcontainers
- **`/cso` 用自评替代,非严格 gstack skill**:用户已授权透明声明模式
- **批量 reorder 并发死锁防御只用 id 升序 update**:高并发下仍可能争锁,MVP 个人用接受
