## 3. 部署步骤(本次仅本地预演)

```bash
# 1) 复制环境变量
cp .env.example .env

# 2) 生成 JWT secret 并写入 .env
openssl rand -base64 48
# Windows PowerShell:
# [Convert]::ToBase64String([System.Security.Cryptography.RandomNumberGenerator]::GetBytes(48))
# 把输出粘到 .env 的 IDEABOX_JWT_SECRET=

# 3) 启动开发数据库
docker compose -f docker-compose.dev.yml --env-file .env up -d

# 4) 等 PG 就绪
until docker exec ideabox-postgres-dev pg_isready -U ideabox; do sleep 1; done

# 5) 跑后端(dev profile,Flyway 自动建表)
set -a && source .env && set +a
mvn spring-boot:run -Dspring-boot.run.profiles=dev

# 6) Smoke test 4 条
curl http://localhost:8080/actuator/health                                    # {"status":"UP"}
curl -X POST -H "Content-Type: application/json" -d '{}' http://localhost:8080/api/auth/login/dev | jq .data.token
# 然后:用 token curl GET /api/users/me 应返用户
# 然后:curl POST /api/projects 创建项目
```

**测试集成跑法**:
```bash
docker exec ideabox-postgres-dev psql -U ideabox -d postgres -c "CREATE DATABASE ideabox_test OWNER ideabox;"
mvn test -DargLine="-Xmx2g -Dfile.encoding=UTF-8"
# 期望:Tests run: 28, Failures: 0, Errors: 0, BUILD SUCCESS
```
