## 3. 部署步骤（本次上线）

```bash
# 1) 服务器准备
ssh ubuntu@<HK_IP>
cd /opt/sosogithub

# 2) 更新 .env.prod（如有新变量）
nano .env.prod   # 加新变量，原有不要误删

# 3) 拉新代码
git pull origin master

# 4) 重建受影响容器（必带 --env-file）
docker compose -f docker-compose.prod.yml --env-file .env.prod up -d --build backend
docker compose -f docker-compose.prod.yml --env-file .env.prod up -d --build frontend
# nginx 改动时：--force-recreate nginx

# 5) Smoke test
docker ps | grep sosogithub-       # 全 healthy
curl -I https://sosogithub.cn/<新接口>
docker logs sosogithub-backend --tail 50 | grep -iE "Started|ERROR"
```
