# IdeaBox · UI 设计规范

> 项目级通用设计规范，所有功能 / 所有端共享。
> 各功能的页面级 UI 规格在对应功能目录的 `01-设计.md § 3` 中，引用本文档的 Token。

---

## 1. 设计原则

| 原则 | 说明 | 体现 |
|---|---|---|
| 极简 | 打开就能用，零学习成本 | 无引导页、无教程、首屏即功能 |
| 对话式 | 像发微信一样记想法 | 底部输入框 + 发送按钮 |
| 层次清晰 | 未完成优先，已完成不干扰 | 已完成折叠/灰显 |
| 快 | 从打开到记录 < 5 秒 | 小程序首屏即最近项目 |

---

## 2. 颜色系统

| Token | 值 | 用途 |
|---|---|---|
| `--primary` | `#1677FF` | 主题蓝，CTA 按钮 / 输入焦点 / 链接 |
| `--bg-primary` | `#FFFFFF` | 主背景 |
| `--bg-secondary` | `#F5F7FA` | 次背景 / 卡片底色 |
| `--text-primary` | `#1D2129` | 主文字 |
| `--text-secondary` | `#86909C` | 辅助文字 / 已完成项文字 |
| `--border` | `#E5E6EB` | 分割线 / 边框 |
| `--success` | `#00B42A` | 完成勾选 |
| `--danger` | `#F53F3F` | 删除操作 |

---

## 3. 字体

| 端 | 字体栈 |
|---|---|
| 小程序 | 系统默认（iOS PingFang SC / Android Noto Sans CJK） |
| 网页端 | `-apple-system, BlinkMacSystemFont, "PingFang SC", "Noto Sans SC", sans-serif` |

## 4. 圆角

| Token | 值 | 用途 |
|---|---|---|
| `--radius-s` | `4px` | 按钮 / 输入框 |
| `--radius-m` | `8px` | 卡片 |
| `--radius-l` | `12px` | 弹窗 / Modal |

## 5. 间距

| Token | 值 |
|---|---|
| `--space-xs` | `4px` |
| `--space-s` | `8px` |

---

## 6. 通用状态设计

| 状态 | 表现 |
|---|---|
| 加载中 | 骨架屏（灰色占位条） |
| 空数据 | 居中插图 + 鼓励性文案 |
| 网络错误 | 顶部 toast "网络异常，请重试" + 重试按钮 |

---

## 7. 文案规范

- 产品名：**IdeaBox**（暂定，英文 wordmark）
- 界面文案：中文
- 按钮文案简短：发送 / 完成 / 取消 / 删除
- 空状态用鼓励性语气："记录你的第一个想法吧"
- 错误提示用中性语气："网络异常，请重试"
- HTTP header / URL 中出现的文案一律 ASCII
